# AI-Powered Personal Finance Assistant & Budget Planner

This project involves developing a personal finance assistant that leverages the power of Generative AI to assist users in managing their finances effectively. The application will provide features such as budget planning, expense analysis, AI-driven financial advice, savings goal planning, and basic investment guidance. The project emphasizes practical application of AI in finance, focusing on building a fully functional system that runs locally without relying on a database.

The application will be built using Python and will integrate with LLM APIs like Gemini or OpenAI for AI-powered features. The project will require students to develop skills in prompt engineering, data visualization, and user interface design. The goal is to create an interactive and user-friendly dashboard that provides real-time financial insights and recommendations.

This capstone project is designed to provide students with hands-on experience in applying AI to solve real-world financial problems. It will enable them to understand the capabilities of Generative AI in the finance domain and develop practical skills in building AI-powered financial tools.
